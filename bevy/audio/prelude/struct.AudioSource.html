Expand description

A source of audio data

Fields§

§bytes: Arc<[u8]>

Raw data of the audio source.

The data must be one of the file formats supported by Bevy (wav, ogg, flac, or mp3). It is decoded using rodio::decoder::Decoder.

The decoder has conditionally compiled methods depending on the features enabled. If the format used is not enabled, then this will panic with an UnrecognizedFormat error.
